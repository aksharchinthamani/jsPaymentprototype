

# esyPay

**esyPay** is a lightweight, web-based payment interface prototype designed to demonstrate a seamless user experience for digital transactions. This project focuses on the front-end flow of handling payments using JavaScript.

##  Live Demo

You can view the live prototype here: [esyPay Live Demo](https://aksharchinthamani.github.io/jsPaymentprototype/) 

##  Features

* **Modern UI/UX**: A clean and intuitive interface for entering payment details.
* **JS Integration**: Built using core JavaScript to handle dynamic interactions.
* **Responsive Design**: Optimized for viewing across different screen sizes.
* **Prototype Flow**: Demonstrates the step-by-step process from input to transaction simulation.

##  Technology Stack

* **HTML5**: Structure and semantics.
* **CSS3**: Styling and layout.
* 
**JavaScript**: Logic and interactivity.
